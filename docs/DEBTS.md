# DEBTS — единый трекер технического долга

Единственный трекер долгов проекта (по канону `.claude/team/engineering-discipline.md` §4).
Любой временный хак, отложенный фикс, красный/скипнутый тест, известная мина — **строка здесь**.
Долг без строки = долга «не существует» для команды и всплывёт в худший момент.

Закрытый долг помечаем ✅ с датой и коммитом, но не удаляем (история).
