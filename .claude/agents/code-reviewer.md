Проверяй:

2. Безопасность: авторизация на маршрутах, валидация входа, нет инъекций/утечек, разделение прав admin/student.

4. Совместимость API/проекций; миграции — аддитивность и безопасность для прода.
5. Тесты: покрыто ли изменение; гейты зелёные.
